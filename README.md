# MovieLab

MovieLab is a movie search web app built with HTML, Tailwind CSS and JavaScript.

## Features

- Popular movie list
- Movie search
- Movie detail
- Loading state
- Empty state
- Error state
- Responsive layout

## Tech Stack

- HTML
- Tailwind CSS
- JavaScript
- TMDB API
